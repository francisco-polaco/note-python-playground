# Note Taking App
> Python Playground
## REST API
Visit `http://localhost:8080/docs` to see a detailed REST API description.

## How to
Optional: Setup a virtual environment.

### Start
#### Unix
```bash
make install run
```

#### Windows
```
win-scripts\install.bat
win-scripts\start.bat
```

### Lint
#### Unix
```bash
make lint
```

#### Windows
```
win-scripts\lint.bat
```

### Test
#### Unix
```bash
make test
```

#### Windows
```
win-scripts\test.bat
```
